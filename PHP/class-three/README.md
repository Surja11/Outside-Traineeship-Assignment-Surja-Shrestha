# Register and Login System
Register and Login System is an assignment to better understand server -side logics and PHP programming language. 
This project consists of a registration page, where user registers themselves. After successful registration, they are redirected to login page. The user logs in with the credentials they provided at the time of registration and then they are redirected to dashboard page. During login, username is stored to session so that the server can remember user. After the user logs out, the session is destroyed and user is sent to login page again.

## Steps to run the project
1. Install XAMPP 
2. Make sure php and mysql are installed.
3. Open xampp and run php and mysql
4. Open http://localhost/phpmyadmin
5. Make a database and create table.
6. Make sure the database user and password are correct.
7. Run the project http://localhost/Outside-Traineeship-Assignment-Surja-Shrestha/PHP/class-three/registration.html
